# Mapeamento Psicológico Totall

Aplicação web estática para coleta e análise inicial de um perfil baseado na Espiral de Valores.

## Funcionalidades

- Identificação por nome, cargo e contexto profissional
- Questionário com 10 blocos e distribuição obrigatória de 12 pontos por bloco
- Matriz final com seis níveis da Espiral de Valores
- Gráfico radar e análise dinâmica do perfil
- Salvamento local do progresso
- Impressão e exportação do resultado em PDF

## Executar localmente

Abra `outputs/perfil-psicologico/index.html` no navegador ou sirva a pasta por um servidor HTTP estático.
